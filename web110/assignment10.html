Learning Outcomes from Week 10

CSS = Cascading Style Sheets
Used to design & decorate webpages
Separates content (HTML) from presentation (CSS)

Syntax:
Selector looks like this {property: value;}

Example:
h1 { color: blue; text-align: center; }
Selector = h1
Property = color, text-align
Value = blue, center

Selectors:
Element - p, h1, body
Class - .classname
ID - #idname
Group - h1, h2, h3
Universal - *

Colors / Backgrounds:
color: red;
background-color: yellow;
background-image: url("img.jpg");

Text / Font:
font-family: Arial;
font-size: 16px;
font-weight: bold;
text-align: center;

Box Model:
Every element = box
Content → Padding → Border → Margin

Units:
px → pixels
em → relative size
% → relative to parent
