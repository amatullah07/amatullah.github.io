Term Summary
VoIP Voice over Internet combined data apps with voice technologies. Phone lines were replaced and people could now use data through internet wires to transfer voice.
Remote Desktop This system lets you access a specific desktop you desire from anywhere. This system helps a lot of WFH workers to access their work computers and even many other IT professionals in the field.
File Transfer and File Transfer Protocol This is the official way to transfer files over a network. There are secured versions available too, such as Secure FTP or FTPS. Earlier versions used to have ftp in the beginning of browser urls.
P2P Peer to Peer is a file sharing protocol where files are shared between two computers without the need of a central server. There are five types of P2P users- creators, sharers, searchers, communicators and freeloaders.
SMS Users type short messages through mobile phones instantly to each other.
Mobile Computing Ability to use technology “on-the-go”
