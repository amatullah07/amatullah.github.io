TutorialsPoint: Internet Reference Models
  • Reference models are standardized methods to help devices communicate with each other.
  • OSI and TCP/IP are two reference models.
  • The TCP/IP model uses four layers whereas the OSI Model has seven layers.
TutorialsPoint: Internet Protocol
  • TCP means transmission control protocol and it is a standard for connections via networks.
  • It helps our data arrive correctly in the right order.
  • It offers six services at the application layer.
  • IP has to pair with a reliable TCP network to ensure correct transmission of data.
  • Other transfer protocols include- UDp, FTP, TFTP, Telnet and HTTP.
TutorialsPoint: Internet Domain Name System
  • DNS is a naming scheme. DNS is like the internet phonebook with easy names for IP addresses.
  • There are two types of URLs- 1) Absolute URL has the complete name. For example: http is the protocol, docs.google.com is the server name and index.htm is the path name. 2)Relative urls are used for internal links like hyperlinks on a wikipedia page.
