Learning Outcome from Lesson 12

Javascript is the behavioral layer of a web page, bringing interactivity and giving it life. Javascript can be added internally (written directly using the script tag) and externally (the code is written as a separate .js file and then linked to the html file). Javascript is usually confused with Java, although there are stark differences in these languages. JavaScript works in every modern browser and basically “talks to” the webpage’s structure (the DOM) so it can change things on the page. It has built-in safety rules so websites can’t peek at stuff they’re not supposed to. It also cares about capitalization, usually uses camelCase for naming things, and its variables can hold any kind of value without you having to declare a specific type.
